Unidad 3


Selección de componentes para ensamble de equipo de cómputo

La selección adecuada de componentes es crucial para construir un sistema de cómputo eficiente y funcional. Esta unidad cubre los principales elementos que conforman un equipo de cómputo moderno.

3.1 Chip set

El chipset es el conjunto de circuitos integrados que gestiona el flujo de datos entre el procesador, la memoria y los periféricos:

3.1.1 Unidad central de procesamiento (CPU)

  • Cerebro del computador: Ejecuta instrucciones y procesa datos.
  • Características clave: Número de núcleos, frecuencia de reloj, caché, arquitectura.
  • Fabricantes principales: Intel (Core i3/i5/i7/i9), AMD (Ryzen 3/5/7/9).

3.1.2 Controlador del Bus

  • Función: Gestiona la comunicación entre la CPU y otros componentes.
  • Tipos de buses: Bus de datos, bus de direcciones, bus de control.
  • Velocidad: Determina el ancho de banda disponible para transferencias.

3.1.3 Puertas de entrada/salida (I/O)

  • Interfaces comunes: USB, Thunderbolt, HDMI, Ethernet, audio.
  • Controladores: Gestionan la comunicación con dispositivos externos.
  • Estándares: USB 2.0/3.0/3.1/4, Thunderbolt 3/4.

3.1.4 Controlador de interrupciones

  • Función: Maneja las señales de interrupción de dispositivos periféricos.
  • Prioridades: Asigna niveles de prioridad a diferentes interrupciones.
  • APIC: Controlador avanzado de interrupciones en sistemas modernos.

3.1.5 Controlador de acceso directo a memoria (DMA)

  • Propósito: Permite a dispositivos acceder a memoria sin involucrar a la CPU.
  • Ventajas: Reduce carga de la CPU, mejora el rendimiento.
  • Canales DMA: Rutas dedicadas para transferencias.

3.1.6 Circuitos de temporización

  • Reloj del sistema: Sincroniza operaciones en la computadora.
  • Oscilador de cristal: Genera la señal de reloj base.
  • PLL: Circuitos de bloqueo de fase para generar múltiples frecuencias.

3.1.7 Circuitos de control

  • Funciones: Generación de señales de control, gestión de energía.
  • Componentes: Decodificadores, multiplexores, flip-flops.
  • Administración térmica: Control de ventiladores y throttling.

3.1.8 Controladores de video

  • GPU integrada: Procesamiento gráfico en el chipset o CPU.
  • GPU discreta: Tarjetas gráficas dedicadas (NVIDIA, AMD).
  • Interfaces: PCIe para tarjetas gráficas, salidas de video.

3.2 Aplicaciones

Componentes fundamentales para el funcionamiento del sistema:

3.2.1 Entrada/salida

  • Dispositivos de entrada: Teclado, mouse, escáner, micrófono.
  • Dispositivos de salida: Monitor, impresora, altavoces.
  • Interfaces: USB, Bluetooth, Wi-Fi, Thunderbolt.

3.2.2 Almacenamiento

  • Tipos: HDD (discos duros), SSD (estado sólido), NVMe.
  • Interfaces: SATA, M.2, PCIe para SSD.
  • Capacidad vs. velocidad: Equilibrio según necesidades.

3.2.3 Fuentes de alimentación

  • Potencia: Medida en vatios (W), según componentes.
  • Eficiencia: Certificaciones 80 Plus (Bronze, Silver, Gold, Platinum).
  • Conectores: ATX 24-pin, EPS 8-pin, PCIe 6+2-pin.

3.3 Ambientes de Servicio

Diferentes contextos donde se implementan sistemas de cómputo:

3.3.1 Negocios

  • Requisitos: Fiabilidad, seguridad, soporte para software empresarial.
  • Componentes típicos: Estaciones de trabajo, servidores, almacenamiento NAS/SAN.
  • Consideraciones: Escalabilidad, redundancia, copias de seguridad.

3.3.2 Industria

  • Entornos difíciles: Resistencia a polvo, vibraciones, temperaturas extremas.
  • Computación industrial: PLC, sistemas embebidos, IoT.
  • Protocolos: Modbus, Profibus, EtherCAT.

3.3.3 Comercio Electrónico

  • Infraestructura: Servidores web, bases de datos, balanceadores de carga.
  • Seguridad: SSL/TLS, firewalls, sistemas de pago seguros.
  • Escalabilidad: Soluciones cloud, autoescalado.
